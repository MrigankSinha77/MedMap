# **App Name**: MedMap

## Core Features:

- Medicine Search: Search for medicines by name, composition, or brand.
- Location-Based Pharmacy Display: Display nearby pharmacies sorted by distance using device GPS.
- Real-Time Availability Status: Show current stock status (available, limited, out of stock).
- Last Updated Timestamp: Display timestamp of the last stock update for transparency.
- Pharmacy Dashboard: Simple UI for pharmacies to update medicine availability.
- Data Reliability Tool: AI tool to flag data if not updated in a timely fashion or according to the update patterns of reliable pharmacies.
- Verified Pharmacy Access: System to ensure only verified pharmacies can list medicines to prevent false information.

## Style Guidelines:

- Choose a dark color scheme. Primary color: Saturated blue (#4285F4), a common 'medical' color.
- Background color: Very light blue (#E8F0FE), for a clean and calm aesthetic.
- Accent color: Purple (#673AB7), for interactive elements and highlights.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern, machined look; suitable for headlines or body text
- Use modern and clear icons to represent medicine types and availability statuses.
- Incorporate a 'liquid glass' effect in the UI similar to iOS for a modern and fluid feel. The interface has a toggle to switch between dark and light mode.
- Subtle animations and transitions to enhance the user experience and provide feedback on interactions.
